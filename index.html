ChatBall

Introduction


ChatBall is a python based tool that perform detailed analyses on WhatsApp chats. Powered by Pandas and python's graph plotting library matplotlib, ChatBall provides beautiful graphs and statistics for you. This website provides information on the ChatBall project and instructions on how to use the tool. The project can be found on GitHub here.


Examples


Some basic statistics that ChatBall provides are given below.

  • Total number of messages sent.
  • Total number of messages sent by individual users.
  • Average length of messages sent by individual users.
  • Most popular emojis sent by individual users.
  • Average number of messages sent by days of the week.
